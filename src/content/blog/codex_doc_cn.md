因为 claude code 被封了，所以最近一段时间都在使用 codex 开发，还开了 pro 的订阅。

但是遇到了一个问题，codex 的文档都是英文版的，没有中文的。（连 claude code 的文档都有中文）。

所以我灵机一动，感觉会有很多人像我一样因为没有办法开通 claude 而去使用 codex，因此如果我做一个 codex 的中文文档，是不是可能会有效果。
